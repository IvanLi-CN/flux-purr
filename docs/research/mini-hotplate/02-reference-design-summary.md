# 02 原版参考设计摘要

本文档总结原版项目（小O和小Q）中对后续二次开发最关键的信息，不展开实现细节。

## 项目定位

- 设备目标：超迷你恒温加热台。
- 供电路径：PD 诱骗 20V 供电。
- 加热面尺寸：5.6 cm x 5.6 cm。
- 交互方式：两按键 + 0.91 寸 OLED。

依据：[S1]

## 已公开硬件基线

- 协议芯片：IP2721（用于 PD 20V 诱骗）。
- 主控芯片：STC8H3K32S。
- 传感器：PT1000（页面硬件说明与评论均提及）。
- 结构连接：控制板与加热板通过 P100 测试针导通。
- 典型功率线索：铝基板约 6 欧，20V 约 65W（原文说明）。

依据：[S1]

## 已公开软件行为基线

- 主界面提供实时温度、设定温度、电压、加热状态、PWM 占空比。
- 操作逻辑：
  - 左右键短按/长按调温。
  - 双键短按切换加热输出。
  - 双键长按进入设置。
- 设置项包括：PID 参数、温度上下限、显示模式（信息模式/曲线模式）。
- 参数保存策略：写入 EEPROM，断电不丢失。

依据：[S1]

## 参考设计口径建议

- 将原版视作“控制逻辑与电路逻辑的主口径”。
- 后续二改若宣称“固件不变”，应先复核其与原版的关键连接关系。
- 对评论区问题（如温度 000、烧录失败）优先归入 Bring-up 风险，不直接改写主设计结论。
