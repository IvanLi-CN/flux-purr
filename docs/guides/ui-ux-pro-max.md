# UI UX Pro Max Usage (Repo-local)

This repository installs UI UX Pro Max under `.codex/skills/ui-ux-pro-max/`.

## Design search workflow

1. Product domain query (device console / embedded admin panel)
2. Style query (stack: React + shadcn)
3. Typography and color query
4. Apply result into component stories before implementation freeze

## Prompt examples

- `Search a style for embedded admin UI with clear telemetry readability`
- `Suggest shadcn-friendly color palette for a light-mode industrial dashboard`

## Delivery checklist

- No emoji icons in production UI
- All interactive controls have visible hover/focus state
- Storybook stories cover default + critical edge state
- Contrast target reaches WCAG AA for text controls
- Mobile width (375px) and desktop width (1440px) verified
